# Tentang

SMS Guard Pro adalah aplikasi manajemen operasional Satpam yang dirancang untuk membantu:

- Danru
- Chief Security
- Supervisor
- Koordinator Area

dalam mengelola seluruh aktivitas operasional secara digital.
